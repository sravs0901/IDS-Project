# IDS-Project
# Intrusion Detection System (IDS)

## 📌 Overview
This project is a Python-based Intrusion Detection System that analyzes datasets and detects anomalies.

## 🚀 Features
- Client-Server communication
- Dataset-based detection
- Machine Learning / Rule-based detection
- LLM integration (if applicable)

## 🛠 Tech Stack
- Python
- Dataset: KDD CUP

## 📂 Project Structure
- src/ → Core code
- dataset/ → Data files
- scripts/ → Run scripts
- docs/ → Screenshots & documentation

## ▶️ How to Run

### Install dependencies
pip install -r requirements.txt

### Run server
python src/Server.py

### Run client
python src/Client.py
